Migrate ClockTime dataclass from v1 to v2 Pydantic config:

2. Replace the inner `class Config:` with `model_config = ConfigDict(frozen=True)`

Note: This is a `@pydantic.dataclass`, not a BaseModel. For dataclasses, the config pattern is slightly different but still uses ConfigDict in v2.

The ClockTime class uses `@dataclass(frozen=True, order=True)` from pydantic. In Pydantic v2, the inner Config class should be replaced with model_config.

Before:
```python
@dataclass(frozen=True, order=True)
class ClockTime:
    ...
    class Config:
        allow_mutation = False
```

After:
```python
from pydantic import ConfigDict

@dataclass(frozen=True, order=True)
class ClockTime:
    model_config = ConfigDict(frozen=True)
    ...
```

Actually, for pydantic.dataclass, the frozen=True in the decorator already handles immutability. The inner Config may be redundant. Remove the class Config entirely or convert to model_config if needed.
