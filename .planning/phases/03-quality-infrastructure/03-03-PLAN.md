  <name>Task 2: Create pre-commit configuration</name>
  <files>.pre-commit-config.yaml</files>
  <action>
Create `.pre-commit-config.yaml`:

```yaml
# Pre-commit hooks for quality gates
# Install: uv run pre-commit install
# Run manually: uv run pre-commit run --all-files

repos:
  # Ruff - fast Python linter and formatter
  - repo: https://github.com/astral-sh/ruff-pre-commit
    rev: v0.9.0
    hooks:
      # Linter - runs ruff check with auto-fix
      - id: ruff
        args: [--fix]

      # Formatter - runs ruff format
      - id: ruff-format
```

Design decisions:
- Use ruff-pre-commit v0.9.0 (stable, matches ruff version)
- Enable --fix for ruff lint to auto-fix issues
- Do NOT include coverage check in pre-commit (too slow for every commit)
- Coverage is enforced in CI instead
</action>
